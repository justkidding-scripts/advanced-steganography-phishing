Getting Started

Welcome to the Telegram Desktop client documentation. This guide will help you understand the client's architecture, security features, and update mechanisms.

Architecture Overview

The Telegram Desktop client is built using:

  • Qt Framework for cross-platform UI
  • OpenSSL for cryptography
  • zlib for compression
  • FFmpeg for media handling

Update System

The client uses a secure update system with several components:

UpdateChecker
├── VersionControl
├── SecurityVerification
└── DeploymentManager

Security Features

Key security features include:

  • End-to-end encryption
  • Perfect forward secrecy
  • Certificate pinning
  • Local database encryption
